ice-auto-select-content-on

Directive Info

To automatically select the content of an (input) element when an event (configurable) is triggered on the element.

Available since

0.1.0

Usage

Needed module (app dependency): ice.forms

        
            <input
            ...
            ice-auto-select-content-on[="mouseover"]
            [ice-deselect-event="mouseout"]
            ...
            </input>

Arguments

Param Value type Default value Details
ice-auto-select-content-on string click Valid event. Value is optional
ice-deselect-event
(optional)		 string blur Valid event.

Example(s)

When default events are used: element selected on 'click' + de-selected on 'blur'

When element selected on 'mouseover' + de-selected on 'mouseout'

ice-bank-account-number

Directive Info

A bank-account-number input field that allows both BBAN and IBAN bank accounts.
Based on the input (BBAN, Belgian IBAN or International IBAN):

  • The field content will be formatted differently
  • Other characters will be filteroud out during input (e.g. BBAN only allows digits)

Available since

0.5.0

Usage

Needed module (app dependency): ice.bank

        
            <input type="text"
            ng-model="someModelVar"
            ...
            ice-bank-account-number
            ng-trim="false"
            style="letter-spacing: 2px;"
            ...
            </input>

Example

See the difference when inputting either "a numeric", "BE" or "an alphabetic" as the first character(s).
