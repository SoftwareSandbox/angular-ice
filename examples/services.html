iceCaret

Service Info

To set the caret (cursor) position of a DOM element. This service is best used from a directive.
The 'setPosition()' code was almost entirely taken from here (thanks again stackoverflow!).

Available since

0.3.0

Usage

        
            angular
            .module('myModule')
            .directive('myDirective', function(iceCaret) {
            return {
            restrict: 'A',
            link: function(scope, element, attrs) {
            ...
            var integerTargetPosition = ...
            iceCaret.setPosition(element[0], integerTargetPosition);
            ...
            iceCaret.setPositionToEnd(element[0]);
            ...
            }
            };
            });
