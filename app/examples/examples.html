Form directives

ice-auto-select-content-on

Directive Info

To automatically select the content of an (input) element when an event (configurable) is triggered on the element.

Usage

        
            <input
            ...
            ice-auto-select-content-on[="mouseover"]
            [ice-deselect-event="mouseout"]
            ...
            </input>

Arguments

Param Value type Default value Details
ice-auto-select-content-on string click Valid event. Value is optional
ice-deselect-event
(optional)		 string blur Valid event.

Example(s)

When default events are used: element selected on 'click' + de-selected on 'blur'

When element selected on 'mouseover' + de-selected on 'mouseout'
