Installation

bower

        
            bower install angular-ice [--save-dev]

--save if you want to use some of the directives, services, etc.
--save-dev if you only need the iceUnit(Tester) and/or the IceBasePage

Setup to use some ice directives, services, ...

  • Add the following <script> to your index.html

    •         
            <script src="bower_components/angular-ice/angular-ice.js"></script>

  • Then add 'ice.forms' as a dependency for your app (or 'ice.common' if you only need some common services)

    •         
            angular.module('myApp', ['ice.forms']);

    Setup to use the iceUnit(Tester)

  • add the following file to your karma.conf.js files array (after the angular modules, but before your own js files):

    •         
            'bower_components/angular-ice/angular-ice.unitTester.js'

  • when using e.g. jshint, add the following to your "globals" in your .jshintrc file (test scope):

    •         
            "iceUnit": false

  • now you can use the globally available (only in karma scope) var iceUnit in your test js files
